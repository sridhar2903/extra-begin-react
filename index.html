REACT FEATURES:

React uses Virtual Dom

React uses JSX,javascript xml

React has react-native for building mobile applications

React has a feature of Components and reusability of components

VIRTUAL DOM:

Virtual DOM is the representation of DOM

Virtual DOM has two copies:A COPY AND B COPY

The changes are only updated in the B copy and then two copies are compared with the help of diffing operation,only the differences found out is applied to the Real DOM

Thus instead of updating the entire page only the changes are being updated in the UI

Differences between Functional and Class Components

Functional component is similar to javascript,which will return the JSX

Class component has a render method inside it ,which will return the JSX

In Functional component,the state can be updated with React hook 'useState'

In Class component,the state can be updated with 'this.Setstate'

Functional Components are mostly preferred because react is a javascript library and so javacript is a functional programming language.

What is single page application

Single page application does not use default method of loading new pages,instead it will update the current page with new data,this is what a Single Page Application means.

Single Page Application is achieved through REACT ROUTING

What is React Routing

React Router is a library with the help of which we can achieve routing in react.

React Router library is categorized into two:react-router-dom:for web applications and react-router-native:for mobile applications.

Routing means moving from one page to another page without reloading the application and thus making it a Single Page Application.

So in our App.js we have to wrap the components with browser router and we have following components in React Router:

Router:which will provide the routing paths

Routes :it is a container in which we have multiple routing paths,whichever is selected that particular component or page will be rendered

Route:corresponds to a single routing path and also a single component,there can be multiple route ,all enclosed between Routes
